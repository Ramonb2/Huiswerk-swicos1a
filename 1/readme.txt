How-To:

You have to use Window Mode in H1Z1

go to your SteamLibary->common->H1Z1->UserOptions.ini -> 
and do this ->
Maximized=1
Mode=Windowed
FullscreenMode=Fullscreen

Start H1Z1
Start External ESP

HotKey's:

INSERT = Show's Settings Menu
POS1(HOME) = Reload Player Object

NUMPAD '+' = Incrase Draw Object Distance ( Player & Zombies )
NUMPAD '-' = Decrase the Distance
NUMPAD '*' = Increase Draw Items Distance
NUMPAD '/' = Decrease the Distance

END = Exit External ESP
