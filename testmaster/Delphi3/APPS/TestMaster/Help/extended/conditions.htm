CONDITIONS OF USE WHEN TESTMASTER IS PUT ON A USER'S WEBSITE

  1. Ensure TestMaster is in Read Only mode (to prevent site visitors from receiving a 'free' copy of the fully enabled TestMaster - with your registration name on it):
    • If Edit/Add/Delete buttons are visible in the main TestMaster window then select the Application | Administrator option, enter your password
    • Uncheck the third option 'Enable question sets Add/Edit buttons'
    • Exit Administrator - the Add/Edit/Delete buttons should be disabled
  2. Distribute the files in a ZIP file including:
    \TestMaster\TestMaster.exe
    \TestMaster\TestMaster.opt
    \TestMaster\Tests\blank.tmq
    \TestMaster\Help\main.htm
    \TestMaster\Help\run.htm
    NOTES:
    • It is important that folder information is included in the ZIP file so that the folder structure is restored when a user extracts the program
    • DO NOT include the extended help files contained in the subdirectory \TestMaster\Help\Extended
    • DO NOT change the names of TestMaster.exe or TestMaster.opt
    • A 100 x 50 pixel graphic (named 'logo.bmp') can be placed in the \TestMaster directory for display alongside the registration information in the About box
  3. Once the files have been zipped, the Add/Edit buttons can be re-enabled.

Many thanks for you cooperation.
