ADMINISTRATOR

This dialog window is reached via the Application | Administrator option, unless Read Only mode has been selected in which case the option does not appear. In that case press Ctrl-A to bring up the dialog box.

This facility is password protected. In the unregistered version the password is 'password'. For a registered copy the password is initially set to the user's registration code, but can be changed as detailed below.

There are various options:

  • Enable answers button - the answers button appears on the run test screen and allows the user to 'preview' the answer to the question. Remove the check in this box to disable this facility.
  • Enable clear results button - this button appears on the Results Summary window. Remove the check in this box if the user should not be able to clear the results file.
  • Enable question sets Add/Edit buttons - removing the check in this box sets 'Read Only' mode and the user cannot edit, add or delete question sets. Setting 'Read Only' mode also hides the Administrator option on the Application menu, and the Register button in the About box.
  • Enable question sets Delete button - if not using 'Read Only' mode but you still want to prevent question sets being deleted, then remove the check in this box.
  • Set test screen colour - allows the background colour of the test screen to be changed.
  • Set default options - restores default options (all cleared).
  • Change Password - when first registered the Administrator password is the same as the registration code of the user. This can be set to something more meaningful with this option. NOTE: There is no request for the original password first since this had to be entered to reach the Administrator window. Do not leave the computer unattended with the Administrator dialog open if there is a possibility someone could change the password !

OK - pressing the OK button implements any changes that have been made.
CANCEL - pressing the Cancel button exits the window without making any changes to the system options.
