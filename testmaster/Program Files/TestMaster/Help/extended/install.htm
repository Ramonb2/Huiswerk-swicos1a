  INSTALLATION

TEST MASTER

The best way to install TestMaster is to unzip the program zip file using the 'Use Folder Names' option in WinZip. This will produce the following folder structure:

\Program Files\TestMaster\

TestMaster.exe
TestMaster.opt

\Program Files\TestMaster\Help

Various help files

\Program Files\TestMaster\Tests

blank.tmq

Putting a shortcut to the TestMaster.exe file on the Start Menu / Desktop completes the installation.

QUESTION SETS

If downloaded question sets are unzipped, again using the 'Use Folder Names' option in WinZip, they will be placed in:

\Program Files\TestMaster\Tests

Note by author: Initially I included an automatic set-up wizard with TestMaster to create all the necessary directories, shortcuts, uninstall logs etc. Unfortunately this increased the zip file to 2MB in size. As the program installation described above is very simple I made the decision that a dramatically smaller download file was preferable. I hope users agree. RCB
