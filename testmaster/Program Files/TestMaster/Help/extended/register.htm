  REGISTRATION

You are encouraged to register this software for a number of reasons:

  • It supports the work that went into it, and encourages further development work
  • It removes the restriction on editing question sets - set at 5 questions per set
  • You get your own name and details in the About box and in the title bar
  • All registrants are entitled to free updates, as they emerge, notified by email

To register the software you need a registration code. This costs:

  • £20.00 sterling (the preferred option)
  • $40.00 US (which includes a $10.00 banking fee)

Email the author for details of where to send the cheque (until I can do it directly via the Net using credit cards) and a registration code will be sent by email.

Once the registration code has been obtained select the Help | About menu option to bring up the About box and press the Register button. Enter the following:

  • Registered name - exactly as given when applying for the registration code.
  • Registration code - exactly as provided.
  • Your email address - optional.
  • Your web address - optional.

Then press 'Submit' to complete registration. If all the details are correct an appropriate message will be shown and all the restrictions will be lifted.

NOTE: Both a user's email and web address can be entered in the registration box for display in the About box. In addition the user can place a 100 x 50 pixel graphic (logo) in the \TestMaster directory for display alongside their registration information in the About box. The graphic should be named 'logo.bmp'.
