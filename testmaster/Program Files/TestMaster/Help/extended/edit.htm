  EDITING A QUESTION SET

Question 0 - Question 0 is not actually part of the question set but is used to enter the summary information seen on the initial TestMaster screen. If the first optional answer is checked then the order of all questions in the set will be randomized when the test is run, if unchecked the question order will be as entered. If a 320 x 50 graphic is provided as an exhibit it will be used as the logo above the summary information in the initial TestMaster screen, and in the run test window.
NOTE: If a logo is provided, the colour of the bar in the run test screen is set to that of the top left pixel of the logo, for continuity.

Starting at the right hand side of the screen:

  • Add - adds a question to the set. Puts the system in edit mode (only Save/Cancel buttons enabled)
  • Delete - deleted displayed question, subject to confirmation.
    NOTE: Question 0 (summary) cannot be deleted
  • Save - saves current changes in the question set, and restores View mode
  • Cancel - cancel any changes to this question, and puts display in View mode
  • Answers: Single / Multiple - Selects the number of permitted responses (Single displays radio buttons as the options, multiple displays check boxes).
  • Navigation buttons - First : Next : Previous : Last. Allows the user to move through the question set.
  • Find - used to search for a phrase within the question set. A dialogue box is displayed where the search text can be entered and the First/Next occurrence searched for. This dialogue box and the edit window can be used at the same time.
  • Protected - selecting this option will prevent anyone with a different registration code from the author from editing the question set
  • Close - stop editing, and return to the main TestMaster screen.

At the left of the screen:

  • Question text - the question being asked
  • Exhibit - a graphic may be included for each question. It should be in BMP format and placed in the Tests sub-directory. Pressing the exhibit button provides a list of all the BMP graphics available. If one is chosen a small representation of it will be shown in the edit window. If a question has an associated exhibit then three buttons will be shown beneath it:
    • Show exhibit - displays the graphic full size in its own window
    • Delete exhibit - removes the reference to the exhibit
    • Change exhibit - displays the list of available graphics for another image to be selected
  • Possible answers:
    • Randomize - if the check box is ticked then the order of the answers will be randomized. If not ticked then the order of the answers will be as entered (useful for those scenario question answers)
    • Optional answer text is entered in the boxes provided (maximum of eight options)
    • The buttons to the left of each answer select the correct answer(s)
  • Reference / Comment - (optional) this is the text shown with the answer when the 'Answer' button is pressed or when incorrect questions are reviewed. The text should explain the answer and any relevant reference(s).
